## Web Project

### Run the application

`npm run build` \
`python .\server\vtkpython\vtk_server.py --port 1234 --content .\dist\` \
`Open your browser to http://localhost:1234/` \
Or you can live development \
`npm run serve & python .\server\vtkpython\vtk_server.py --port 1234` \
`Open your browser to http://localhost:1234/`
